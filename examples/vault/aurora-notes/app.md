## Store Listing (en)

### App Name

### Short Description

### Full Description
Aurora Notes is a fast, beautiful markdown editor that keeps your notes on your device — no account required, fully usable offline.

■ Write in Markdown
Clean live preview, syntax highlighting, and keyboard shortcuts that stay out of your way.
